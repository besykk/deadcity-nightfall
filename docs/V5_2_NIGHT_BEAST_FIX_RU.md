# Dead City: Nightfall v5.2 Night Beast Fix

Что исправлено и добавлено:

- Beast Instinct больше не просто декоративная полоска. Она копится за убийства заражённых.
- За сильных врагов даётся больше энергии: обычные заражённые меньше, ночные прыгуны больше, альфа-прыгун больше всего.
- Добавлены специальные добивания. Они срабатывают по низкому HP врага, при спринте/шифте/ударе сверху, либо когда Beast Instinct готов.
- При готовом Beast Instinct добивание активирует короткий режим зверя: сила, скорость, сопротивление, ночное зрение.
- Night Volatile получил больше HP и урона.
- Добавлены два новых вида ночных прыгунов:
  - Volatile Stalker — быстрый охотник для среднего сектора города.
  - Armored Volatile — живучий бронированный прыгун для дальнего/опасного сектора.
- Ночной спавн теперь учитывает сектор карты: чем дальше от центра, тем опаснее варианты прыгунов.
- Исправлена последняя ошибка со звуком Noise Maker.
- Текстуры мобов и важных 3D-предметов стали ярче и читаемее.

Проверка в игре:

```mcfunction
/gamemode creative
/time set night
/summon deadcity:night_volatile
/summon deadcity:volatile_stalker
/summon deadcity:armored_volatile
/summon deadcity:alpha_volatile
```

Проверка Beast Instinct:

1. Убей несколько заражённых.
2. Смотри на полоску Beast Instinct сверху.
3. Когда враг почти мёртвый, атакуй его в спринте/с шифтом/после прыжка.
4. Должна появиться надпись `SPECIAL FINISHER`, а полоска должна пополниться.
