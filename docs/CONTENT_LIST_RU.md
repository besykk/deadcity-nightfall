# Dead City: Nightfall v3.0 Ultimate Alpha — список контента

## UV / выживание

- UV Flashlight — ПКМ держать, светит по зараженным.
- Battery Cell — заряжает UV Flashlight.
- UV Flare — одноразовая УФ-вспышка по площади.
- Medkit — сильное лечение.
- Bandage — слабое лечение + регенерация.
- Stamina Booster — скорость, прыжок и haste на время.
- Lockpick — пока ресурс для будущих locked контейнеров.
- Noise Maker — приманка, заставляет зараженных побежать к игроку.
- Throwing Knife — метательный нож-placeholder.

## Огнестрел

Чтобы стрелять: ПКМ.  
Чтобы перезарядить: присесть + ПКМ.

- Makeshift Pistol — 6 патронов, слабый.
- Police Pistol — 12 патронов, стабильнее.
- SMG — 24 патрона, быстрый cooldown.
- Pump Shotgun — 6 патронов, стреляет несколькими projectile.
- Hunting Rifle — 5 патронов, высокий урон.

Патроны:

- 9mm Ammo.
- Shotgun Shells.
- Rifle Rounds.

## Машина / The Following-style

Пока это технический prototype:

- Buggy Controller — предмет-ключ. ПКМ включает/выключает двигатель.
- Пока двигатель включен и контроллер выбран в руке, игрок получает скорость и прыжок.
- Fuel Can заправляет Buggy Controller.
- Buggy Repair Kit, Buggy Parts, Car Battery, Reinforced Tires, Engine Parts — база для будущей настоящей машины.

Следующая версия должна заменить prototype на настоящую `ScoutBuggyEntity` с Blockbench-моделью.
