# Dead City: Nightfall v5.0 Release — список систем

## UI / HUD
- UV battery HUD.
- Ammo HUD.
- Weapon durability HUD.
- Current weapon slot HUD.
- Night danger bar.
- Beast Instinct bar.

## Parkour
- Parkour Harness: скорость, прыжок, смягчение падения.
- Grappling Hook: притягивание к блоку по ПКМ.
- Glider Rig: удерживай Shift при падении, чтобы планировать.
- Beast Injector: временный усиленный режим.

## Weapons
- Одноручное оружие.
- Двуручное оружие.
- Электро/огонь/UV/bleed варианты.
- Огнестрел с магазинами, патронами и перезарядкой.
- Прочность через стандартную систему maxDamage.

## Vehicle
- Scout Buggy Deploy Kit.
- Rideable buggy entity.
- Fuel Can для заправки.
- Repair Kit для ремонта.
- UV headlights effect.
- Апгрейд-предметы: spiked bumper, offroad engine, UV core, turbocharger, armor plates, suspension, nitro.
