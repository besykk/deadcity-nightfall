# Dead City: Nightfall v4.0 — список систем

## Игрок

- Паркурная система через Parkour Harness.
- Крюк-кошка: Grappling Hook тянет к блоку.
- УФ-фонарь с батарейкой.
- HUD: батарея, патроны, ночь, крюк.

## UV

- UV Flashlight.
- UV Flare.
- UV Trap.
- UV Safe Zone Beacon.
- UV Headlights для багги как апгрейд-предмет.

## Машина

- Scout Buggy Deploy Kit спавнит rideable Scout Buggy.
- Fuel Can заправляет.
- Buggy Repair Kit ремонтирует.
- Есть предметы для будущих апгрейдов: турбина, броня, подвеска, UV-фары, нитро.

## Зараженные

- Infected Biter.
- Night Volatile.
- Ночные баффы.
- UV-слабость.
- Лут с зараженных.

## Выживание / прогрессия

- Survivor Radio: мини-квест на сдачу ресурсов.
- Safehouse Kit / Quest Board / Loot Cache / Faction Token как база под систему города, фракций и safe zone.

## Следующее развитие после v4.0

- Полноценная генерация структур города.
- Настоящие NPC и диалоги.
- Кастомные 3D-модели мобов через GeckoLib или собственный renderer.
- Кастомная физика багги и отдельная модель вместо vanilla boat renderer.
