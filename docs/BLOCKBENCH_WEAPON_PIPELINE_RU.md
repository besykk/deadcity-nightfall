# Как дальше делать оружие в Blockbench

## 1. Открыть модель

Открываешь:

```text
blockbench/realistic_assets/reinforced_machete.bbmodel
```

или любой другой файл из этой папки.

## 2. Какие части лучше делать кубами

### Оружие ближнего боя

- рукоять — длинный тёмный куб;
- гарда — короткий куб поперёк рукояти;
- лезвие — 2–4 вытянутых куба;
- остриё — маленький узкий куб;
- модификация — отдельные тонкие кубы: провода, катушка, UV-полоска, огненный баллон.

### Огнестрел

- receiver / корпус — главный прямоугольник;
- barrel / ствол — длинный тонкий куб;
- grip / рукоять — куб под углом;
- magazine / магазин — куб снизу;
- stock / приклад — куб сзади;
- scope / прицел — маленький куб сверху.

## 3. Как наносить текстуру

1. В Blockbench открой вкладку Paint.
2. Выбери нужный куб.
3. Крась стороны отдельно:
   - металл — серый/тёмный;
   - рукоять — чёрный/коричневый;
   - UV — голубой;
   - электричество — жёлтый;
   - огонь — оранжевый/красный.
4. Не делай всё одним цветом: добавь царапины, тёмные углы, светлые края.

## 4. Как сохранить в мод

Minecraft item model:

```text
src/main/resources/assets/deadcity/models/item/item_name.json
```

Texture:

```text
src/main/resources/assets/deadcity/textures/item/item_name.png
```

Потом запускаешь:

```text
Gradle → Tasks → fabric → runClient
```
