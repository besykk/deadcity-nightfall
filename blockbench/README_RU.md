# Blockbench templates

В этой папке лежат черновики для Blockbench. Сейчас игровые 3D item-модели уже находятся в:

```text
src/main/resources/assets/deadcity/models/item/
```

Эти JSON можно открыть или пересобрать в Blockbench как Java Block/Item модели. Следующий нормальный шаг — сделать красивые `.bbmodel` исходники для:

1. UV Flashlight.
2. Reinforced Machete.
3. Fire Axe.
4. Makeshift Pistol.
5. Pump Shotgun.
6. Scout Buggy.

Потом экспортировать их обратно в JSON/PNG.
